# clean-cms

This is a quick clean-cms implementation aimed at providing a streamlined content management system solution.

## Features
- Simple and clean architecture
- Easy to set up and deploy

## Tech Stack
- Node.js
- Express
- MongoDB

## Usage
1. Clone the repository.
2. Install dependencies: `npm install`
3. Start the application: `npm start`
